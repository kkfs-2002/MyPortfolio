Project Overview

A modern, full-stack food ordering platform with real-time tracking and secure authentication

Category Full Stack Web App
Year 2025
Status Completed

About This Project

This is a full-featured Food Delivery Platform developed using the MERN stack (MongoDB, Express.js, React.js, Node.js) with TypeScript for enhanced type safety. The system enables customers to browse restaurants, place orders, track deliveries in real-time, and manage their profiles securely.

The application includes separate dashboards for customers, restaurant owners, and admin users. Features include secure JWT authentication, real-time order tracking via WebSockets, restaurant menu management, and an intuitive checkout process with multiple payment options.

Project Objectives:

  • Build a scalable food ordering platform with MERN stack + TypeScript
  • Implement secure user authentication and role-based access control
  • Create real-time order tracking system using Socket.io
  • Develop restaurant management dashboard for menu and order handling
  • Implement seamless checkout with cart management
  • Build comprehensive admin panel for platform oversight

The system follows modern development practices with clean code architecture, type safety through TypeScript, and optimized database queries for performance. RESTful APIs are used for seamless communication between frontend and backend.

Database Schema (MongoDB)

The system uses MongoDB with Mongoose ODM for flexible data modeling. Key collections include:

User
Collection
{ _id, name, email, password, role, address, phone, createdAt }
Restaurant
Collection
{ _id, ownerId, name, description, cuisine, address, rating, menuItems[] }
Order
Collection
{ _id, userId, restaurantId, items[], total, status, paymentStatus, trackingId }
Review
Collection
{ _id, userId, restaurantId, rating, comment, images[], createdAt }

REST API Endpoints

The application provides comprehensive RESTful APIs with TypeScript type definitions.

POST /api/auth/register
User registration with email verification
POST /api/auth/login
JWT authentication and token generation
GET /api/restaurants
Fetch all restaurants with filters and pagination
POST /api/orders
Create new order with payment processing
GET /api/orders/track/:id
Real-time order status via WebSocket
PUT /api/orders/:id/status
Update order status (restaurant/admin)
